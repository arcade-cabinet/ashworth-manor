# Parlor — Connections

## Exits

### 1. To Foyer (South)

| Property | Value |
|----------|-------|
| Target | `foyer` |
| Type | `door` |
| Position | (0, 1.5, -5) |
| Locked | No |

This is the ONLY exit. The parlor is a dead end — one way in, one way out. This creates:
- A sense of enclosure (Lady Ashworth's sealed world)
- The diary page discovery feels more impactful because you can't immediately run away
- The dying fireplace behind you, the door ahead — you must walk away from warmth
