## Narrative Purpose

The last supper. Three guests died within the week of the final dinner party. The table is still set. One chair pushed back — someone left in a hurry. Or was pulled away.

- **Death at dinner** — the final gathering before everything ended
- **The frozen moment** — food, wine, candles all stopped mid-event
- **Helena Pierce** — one of the doomed guests (connects to guest room)
- **The violence** — this wasn't gentle. Chairs pushed. Wine spilled. Something happened here.
