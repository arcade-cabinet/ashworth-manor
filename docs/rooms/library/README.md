# Library

**Room ID:** `library`
**Floor:** Upper (+1)

---

## Narrative Purpose

Knowledge corrupted. The library holds three critical game elements:
1. **The globe** — contains the attic key (puzzle solution)
2. **Rituals of Binding** — the occult book explaining what was done to Elizabeth (pickable item for counter-ritual)
3. **The family tree** — fourth name scratched out: E_iza_eth

This room transforms the player from passive explorer to active puzzle-solver. After reading the diary and finding the globe, they have the key. After taking the binding book, the room itself briefly reacts, confirming that the knowledge here is not inert.
